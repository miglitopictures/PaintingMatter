# PaintingMatter
 A simple painting program with some CA explorations made with Godot 4.2!


Commands:
- SIMULATE -> SPACEBAR
- NEXT FRAME -> RIGHT_ARROW
- DRAW  -> LEFT_CLICK
- ERASE -> RIGHT_CLICK
- RESIZE -> MOUSE WHEEL
- CHANGE MODE -> TAB
- BOX SELECT -> SHIFT+MOUSE DRAG
- SOLIDIFY -> "Y"
- CLEAR CANVAS -> DELETE
- TOGGLE HISTORY -> "H"
- CLEAR HISTORY -> CTRL + "H"
- SHOW INSTRUCTIONS -> ESCAPE
- SAVE IMAGE (.PNG) -> CTRL + "S"
